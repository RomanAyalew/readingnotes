 * ID attriibute
 *The HTML id attribute is used to specify a unique id for an HTML element.*
 `<!DOCTYPE html>
<html>
<head>

#newHeader {
  background-color: lightblue;
  color: blue;
  padding: 50px;
  text-align: center;
}
</style>
</head>
<body>

<h1 id="newHeader">new Header</h1>

</body>
</html>
