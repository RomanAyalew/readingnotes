## CSS

* CSS is the language useed to style an HTML document.

* CSS describes how HTML elements should be displayed.
  
  *example;*

`body {`

  `background-color: black;`
`}`

`h1 {`

 ` color: blue;`

  `text-align: center;`

`}`

`p {`

  `font-family: verdana;`

  `font-size: 50px;`

`}`
